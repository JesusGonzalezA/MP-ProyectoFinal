Imagen::Imagen ( const Secuencia &  pixeles)

Constructor. Crea una imagen a partir de una secuencia que contiene los píxeles. Estima las filas y columnas.

Nota
La estimación de filas y columnas se calcula como la combinación que minimiza V(F,C), o la media de las desviacio- nes (al cuadrado) entre cada dos filas consecutivas.
Parámetros
pixelesSecuencia de píxeles
